# Aplicaciones avanzadas en ciencia de datos

## Rúbrica de evaluación para evidencia final

1. Síntesis del problema y del enfoque analítico:

    - Presenta de forma clara el problema de negocio de NovaCredit, el objetivo del análisis y el enfoque analítico general utilizado. Incluye el contexto mínimo necesario para la toma de decisiones, considerando el riesgo de default y la población o segmentación analizada.

2. Comunicación de hallazgos clave y uso de visualizaciones:

    - Selecciona hallazgos clave y los comunica mediante visualizaciones adecuadas. Cada visualización incluye título, variable o métrica representada y una interpretación que refiere explícitamente a un resultado —métrica o segmento— relacionado con el riesgo de default o la segmentación.

3. Narrativa ejecutiva y coherencia del mensaje:

    - Construye una narrativa ejecutiva con una secuencia lógica que integra problema, análisis, resultados e implicaciones para la toma de decisiones. Vincula de manera explícita los resultados obtenidos con el problema de negocio planteado.

4. Recomendaciones y toma de decisiones:

    - Propone recomendaciones claras; cada una incluye la acción sugerida, el hallazgo que la respalda —métrica o segmento— y el efecto esperado para NovaCredit.

5. Claridad, estructura y calidad del paquete ejecutivo:

    - Entrega un paquete ejecutivo —presentación y resumen narrativo— con una estructura definida que incluye resumen, hallazgos, recomendaciones y cierre. Se identifican con claridad los resultados y las conclusiones para una audiencia no técnica.
